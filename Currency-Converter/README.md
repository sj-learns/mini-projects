# Currency Converter 💱

A currency converter web app built with HTML, CSS, and JavaScript.

## Features
- Convert between a wide range of world currencies
- Live exchange rates fetched from the [Fawaz Ahmed Currency API](https://github.com/fawazahmed0/exchange-api)
- Country flags update dynamically based on selected currency
- Swap currencies instantly with the exchange button

## What I learned
This was my first time working with APIs in JavaScript. I learned how to fetch data asynchronously using `async/await`, parse JSON responses, and dynamically update the DOM based on API data.

## Tech Stack
- HTML, CSS, JavaScript
- [Fawaz Ahmed Currency API](https://github.com/fawazahmed0/exchange-api)
- [Flags API](https://flagsapi.com)
